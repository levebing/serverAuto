# serverAuto - 服务器自动巡检系统

## 项目介绍
serverAuto是一个基于Flask的服务器自动巡检系统，用于定期检查服务器的运行状态，包括CPU使用率、内存使用情况、磁盘使用情况等，并生成巡检报告。

## 功能特性

### 核心功能
- **服务器管理**：添加、编辑、删除服务器，支持密码和SSH秘钥两种认证方式
- **分组管理**：对服务器进行分组管理，便于分类和筛选
- **定时任务**：设置定时巡检任务，自动执行服务器巡检
- **巡检记录**：查看历史巡检记录，了解服务器状态变化
- **巡检报告**：生成详细的巡检报告，支持下载为Word文档

### 新增功能
1. **分页功能**：
   - 服务器列表、巡检记录、定时任务、巡检报告页面默认每页显示10条记录
   - 超过10条的记录通过翻页跳转
   - 分页控件样式统一，支持上一页、下一页和页码跳转

2. **分组管理**：
   - 添加服务器界面支持选择分组
   - 服务器列表界面显示服务器分组
   - 筛选条件支持分组筛选
   - 分组下拉列表样式优化，背景色与界面风格一致

3. **界面优化**：
   - 系统设置下的分组管理二级菜单只在鼠标悬停时显示
   - 子菜单显示位置从右侧弹出改为下方显示，布局更合理
   - 分页按钮样式统一，视觉效果更好

## 技术栈
- **后端**：Flask
- **数据库**：SQLite
- **前端**：HTML, CSS, JavaScript
- **模板引擎**：Jinja2
- **巡检工具**：自定义ServerInspector类

## 安装说明

### 环境要求
- Python 3.6+
- pip

### 安装步骤
1. 克隆项目到本地
   ```bash
   git clone <项目地址>
   cd serverAuto
   ```

2. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```

3. 启动服务
   ```bash
   python app.py
   ```

4. 访问系统
   打开浏览器，访问 `http://127.0.0.1:5000`

## 使用方法

### 添加服务器
1. 点击左侧菜单的"服务器列表"
2. 点击"+ 添加服务器"按钮
3. 填写服务器信息，包括服务名称、IP地址、端口、用户名、密码等
4. 选择服务器分组（可选）
5. 点击"确认提交"按钮

### 执行巡检
1. 在服务器列表中，点击对应服务器的"巡检"按钮
2. 等待巡检完成，查看巡检结果
3. 可以点击"下载报告"按钮下载巡检报告

### 设置定时任务
1. 点击左侧菜单的"定时任务"
2. 点击"添加定时任务"按钮
3. 选择服务器、设置执行时间（cron表达式）
4. 点击"确认提交"按钮

### 查看巡检记录
1. 点击左侧菜单的"巡检记录"
2. 可以按服务器分组和IP地址进行筛选
3. 查看历史巡检记录的详细信息

### 管理服务器分组
1. 点击左侧菜单的"系统设置" -> "分组管理"
2. 可以添加、编辑、删除服务器分组
3. 可以调整分组的显示顺序

## 注意事项
- 本系统仅用于内部网络环境，不建议在公网直接部署
- 建议定期备份数据库文件（servers.db）
- 使用SSH秘钥认证时，确保秘钥文件的权限设置正确
